Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install number-formatter-mk-test`

## Usage

    var numFormatter = require('number-formatter-mk-test');

    var formattedNum = numFormatter(696969);
  
  
  Output should be `696,969`


## Tests

  `npm test`
